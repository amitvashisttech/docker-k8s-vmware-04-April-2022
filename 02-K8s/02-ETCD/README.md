# K8s Datastore - ETCD

## Check the status of ETCD
```
kubectl get pods -n kube-system | grep -i  etcd
```

## Let explore ETCD POD
```
kubectl exec -it etcd-kmaster -n kube-system   -- /bin/sh
```
